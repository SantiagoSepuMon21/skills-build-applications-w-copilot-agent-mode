# OctoFit Tracker (scaffold)

Ports:
- Frontend: 5173
- Backend: 8000
- MongoDB: 27017

This repository contains a minimal scaffold for the OctoFit Tracker multi-tier app.

Frontend: React 19 + Vite (in `octofit-tracker/frontend`).
Backend: Node.js + Express + TypeScript + Mongoose (in `octofit-tracker/backend`).
